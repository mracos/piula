**Free Medication Schedule Calculator**

A clean, modern medication scheduler that helps you plan pill-taking times with customizable intervals. Perfect for antibiotics, birth control, and daily medications.

## Features

- **Custom Intervals**: Set any hourly interval for your medication schedule
- **Multi-day Planning**: Plan schedules from 1 to 30 days
- **Daily Schedule View**: See all medication times for a single day at a glance
- **Calendar Overview**: Visualize your entire medication schedule across multiple days
- **Real-time Updates**: Schedule updates instantly as you adjust settings
- **Clean Interface**: Modern, gradient design with intuitive controls
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript - no frameworks needed

## Usage

1. **Set start time**: Choose when you want to take your first dose
2. **Set interval**: Enter how many hours between each dose (e.g., 8 hours)
3. **Set duration**: Choose how many days you need to take the medication
4. **View schedule**: See your daily times and full calendar overview

## Quick Start

Simply open `index.html` in any modern browser. No build process, no installation required.

```bash
git clone <repository-url>
cd piula
open index.html
```
